ZAD SSR FRONTEND APPLICATION STRUCTURE
Audited: 2026-06-10

Purpose
This file documents the ZAD Hotels public SSR website in:
D:\JannatBooking\zad_ssr

The app is a Next.js website for zadhotels.com. It is intentionally not a
Create React App clone. It reads public Zad-scoped hotel data from the
hotels_backend Zad API routes and renders public booking/discovery pages.

Runtime
- Framework: Next.js 16 app router.
- React: 19.
- UI library: Ant Design for form controls, drawers, badges, checkout form, and search inputs.
- Local cart: browser localStorage key zadHotelsCart.
- Language preference: browser localStorage key zadHotelsLanguage.
- Local port: 3105.
- Default API base: https://xhotelpro.com/api unless NEXT_PUBLIC_API_URL or API_URL is set.
- Local commands:
  - npm install
  - npm run dev
  - npm run build
  - npm run start

      listen 80;
      server_name zadhotels.jannatbooking.com;

      client_max_body_size 100M;

      location / {
          proxy_pass http://127.0.0.1:3105;
          proxy_http_version 1.1;
          proxy_set_header Upgrade $http_upgrade;
          proxy_set_header Connection "upgrade";
          proxy_set_header Host $host;
          proxy_set_header X-Real-IP $remote_addr;
          proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
          proxy_set_header X-Forwarded-Proto https;
          proxy_cache_bypass $http_upgrade;
      }
  }
